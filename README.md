# Covid19_API
 Covid19 API를 이용한 국내외 Covid19 현황을 알아 볼 수 있는 프로그램입니다.

## 프로젝트 정보
### 설치
- Visual Studio

### 사용 방법
- Visual Studio를 이용하여 Covid19.sln 파일을 실행 
- Ctrl + F5
